- 코발트가 추가되었습니다. 이제 코발트 유저들 역시 아디나와 함께할 수 있습니다.
- 프롬프트가 대규모로 수정되었습니다. 아디나와 비앙카는 이제 좀 더 똑똑해져야 합니다.
- 사이드 바와 패치 노트 페이지의 디자인 오류를 수정했습니다.
- 공지 사항을 빠르게 전달할 전광판이 신설되었습니다.
- 히든 뱃지가 추가되었습니다. 이 뱃지는 뱃지 조건 페이지에서 찾을 수 없습니다. 모험을 통해 알아가시길 바랍니다.
- 메인 페이지에서 로딩 시 다크 모드 상태일 때 '데이터를 로딩 중입니다' 라는 문구의 글씨가 잘 보이지 않는 이슈가 수정되었습니다.
- 다크 모드 사용시 메인 페이지에서 검색창이 글자가 보이지 않는 오류가 수정되었습니다. 소중한 제보에 감사드립니다.
