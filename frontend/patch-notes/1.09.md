* 오랜 점검에서 수정구슬이 복구되었습니다. 부득이하게 서비스를 일시로 중단하게 되어 사죄의 말씀을 드립니다.

* 평균 드론 사용 횟수와 평균 emp 사용 횟수가 추가되었습니다.

* 내부 로직이 대규모로 수정되었습니다만, 사용자 여러분들께서는 이전과 동일하게 사용하실 수 있습니다. 

* 비교용 지표 통계 수집 방식이 기존과는 완전히 달라졌습니다. 편향 문제가 해결되고, 수집 방식의 한계를 벗어나 이제 저티어 여러분들께서도 티어별 지표를 제공해드릴 수 있게 되었습니다. 추가적으로 기존 방식에선 불가능했던 일반 게임이나, 코발트의 데이터 역시 제공이 가능해져 추가가 될 예정입니다.

* 루트 검색 기능이 완전히 망가졌습니다만, 이는 api 상의 문제로 인하여 해결할 수가 없는 부분입니다. 해결되는대로 수정해보겠습니다.
